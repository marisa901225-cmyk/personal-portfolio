# MyAsset Portfolio 사용설명서

> 개인 자산 관리 + 지출 기록 + 뉴스 자동화를 한 번에

---

## 🎯 이게 뭐 하는 건데?

- **자산 관리**: 주식, 달러, 코인 등 내 자산 시세 자동 업데이트
- **지출 기록**: 스마트폰 알림으로 카드 사용 자동 기록
- **뉴스 요약**: 관심 키워드 뉴스 수집해서 요약본 제공
- **텔레그램 알림**: 아침마다 브리핑 받기

---

## ⚡ 5분 만에 시작하기

### 1단계: 환경변수 설정

`backend/.env`에는 비밀 아닌 설정만 넣고, 토큰/API 키는 레포 밖 `~/ai-models/myasset.secrets.env`에 둠.

```bash
cp backend/.env.example backend/.env
cp backend/.env.secrets.example ~/ai-models/myasset.secrets.env
```

`backend/.env` 예시:

```bash
LLM_BASE_URL=http://openvino-server:8082
ALARM_SUMMARY_LLM_BASE_URL=http://openvino-server:8082
ALARM_RANDOM_LLM_BASE_URL=http://llama-server-vulkan-huihui:8083
KIS_CONFIG_DIR=/root/KIS/config
TRADING_ENGINE_SWING_CASH_RATIO=0.80
TRADING_ENGINE_DAY_CASH_RATIO=0.20
```

`~/ai-models/myasset.secrets.env` 예시:

```bash
JWT_SECRET_KEY=아무거나긴문자열
API_TOKEN=비상시용토큰

# 네이버 로그인용
NAVER_CLIENT_ID=네이버개발자센터에서받은ID
NAVER_CLIENT_SECRET=네이버개발자센터에서받은시크릿

# 선택 (필요하면)
TELEGRAM_BOT_TOKEN=텔레그램봇토큰
TELEGRAM_CHAT_ID=내텔레그램ID
```

기본 secrets 경로를 바꾸고 싶으면 호스트에서 `MYASSET_SECRETS_ENV_FILE=/custom/path/myasset.secrets.env`를 설정하면 됨.

### 2단계: 실행

```bash
docker compose up -d
```

끝. 이제 `http://localhost:5173` 접속

## 🔐 로그인은 어떻게?

**기본**: 네이버 로그인 → 쿠키 자동 저장 → 계속 로그인 유지

**비상용**: API 토큰으로 직접 인증 (서버 문제 있을 때만)

---

## 📱 주요 기능별 사용법

### 1. 지출 기록

**방법 1**: 스마트폰 알림 자동 연동
- 카드 결제 알림이 서버로 전송됨
- 자동으로 지출 기록 생성
- 텔레그램으로 확인 메시지 받음

**방법 2**: 엑셀/CSV 직접 업로드
- 카드사에서 다운받은 파일 업로드
- 중복 자동 제거
- 카테고리 자동 분류

### 2. 자산 관리

- 보유 주식/코인 등록하면 시세 자동 업데이트
- 평일 15:35에 국내 주식 시세 갱신
- 새벽 6:30에 미국 주식 시세 갱신
- 원화 기준으로 통일해서 보여줌

### 3. 뉴스 브리핑

- 10분마다 뉴스 자동 수집
- AI가 중요한 부분만 요약
- 아침 7시~밤 10시 사이 5분마다 텔레그램 알림

---

# 재시작
docker compose restart

# 재빌드
docker compose up -d --build

# 로그보기
docker compose logs -f backend

# 전체 리셋
docker compose down -v && docker compose up -d

# 프론트 배포
npx vercel --prod

## 💡 자주 묻는 질문

**Q. 로그인이 안 돼요**
- 네이버 개발자센터에서 Client ID/Secret 확인
- Redirect URI가 `http://localhost:5173/auth/callback` 맞는지 확인

**Q. API 401 에러 나요**
- 쿠키 만료됨 → 다시 로그인
- 또는 비상용 API 토큰 사용

**Q. 시세가 안 업데이트돼요**
- 한국투자증권 API 설정 확인 (`KIS_CONFIG_DIR`)
- 장 마감 시간 맞는지 확인

**Q. 텔레그램 알림이 안 와요**
- `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID` 확인
- 봇이 차단되지 않았는지 확인

|------|------|------|

| 미국 시세 갱신 | 화~토 06:30 | 나스닥/S&P500 |
| DB 백업 | 매일 06:30 | 자동 백업 생성 |
